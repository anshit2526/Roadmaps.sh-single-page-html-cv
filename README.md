# Roadmaps.sh-single-page-html-cv

A single-page HTML/CSS resume for a Junior Frontend Developer.

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- No additional dependencies required

### Running the Project

#### Option 1: Direct File Opening (Simplest)
1. Open the `index.html` file directly in your web browser by:
   - Double-clicking the `index.html` file, or
   - Right-clicking and selecting "Open with" â†’ your preferred browser

#### Option 2: Using a Local Server (Recommended for Development)

**Using VS Code:**
1. Install the "Live Server" extension
2. Right-click on `index.html` and select "Open with Live Server"

Once running, open your browser and navigate to:
- `http://localhost:8000` (for Python)
- `http://localhost:8080` (for http-server)
- The URL shown by Live Server

## Project Structure
- `index.html` - Main CV content and structure
- `styles.css` - Styling for the resume
- `README.md` - This file
